| Feature                      | File                                 | Status         |
| ---------------------------- | ------------------------------------ | -------------- |
| Recipe Management            | [recipes.md](recipes.md)             | 🚧 In progress |

## Status Legend

| Badge          | Meaning                               |
| -------------- | ------------------------------------- |
| 📋 Defined     | Requirements written, not yet started |
| 🚧 In progress | Currently being implemented           |
| ✅ Done        | Implemented and verified              |

## How to Use This Folder

- Each file covers one feature area.
- Requirements are identified by a unique ID (e.g. `RCP-01`) for easy reference in commits and tasks.
- When a requirement is implemented, update its `Status` field — do not delete it.
